# Smart Support UA Pro v1

**Професійний агент підтримки клієнтів українською мовою з контекстною пам'яттю**

Цей шаблон дозволяє запустити розумного AI-агента, який може:
- Відповідати клієнтам природною українською мовою
- Пам'ятати контекст попередньої розмови
- Класифікувати запити (питання, скарга, статус замовлення)
- Використовувати інструменти (FAQ, статус замовлення, збереження лідів)
- Автоматично ескалувати складні запити на менеджера

### Швидкий запуск

1. Встав свій Anthropic API ключ у файл `docker-compose.yml`
2. Запусти команду:

```bash
docker compose up -d
